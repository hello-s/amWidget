## InputFormatter

对文本框中输入的数据自动格式化为某种固定格式。  


### 示例代码
html
```
	<input type="text" data-format="4,"/>		input内容自动每4个分为1组,分隔符为逗号
```

js
```javascript
	var list = document.getElementsByTagName("input");	//获取输入框元素集
	AW.inputFormatter.listen(list);					//list为要监听的元素以数组形式的集合
```
